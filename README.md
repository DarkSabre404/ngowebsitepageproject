# 🌱 Still Rooted

**Still Rooted** is a responsive, multi-page web platform dedicated to empowering senior citizens. Unlike traditional community programs that look at elders solely as beneficiaries, Still Rooted treats our older adults as leaders, mentors, and educators, providing them a stage to conduct events, pass down invaluable life skills, and stay deeply integrated into society.

Live Site: [https://darksabre404.github.io/ngowebsitepageproject/](https://darksabre404.github.io/ngowebsitepageproject/)

---

## 🚀 Features

- **Dynamic Navigation Ribbon:** A unified header menu (`Home`, `About Us`, `Events`, `Our Team`) with active-state highlights for seamless page transitions.
- **Responsive Double-Card Layout:** An optimized grid system designed to elegantly split structural content into accessible components like "How It Works" and "Why We Do This".
- **Dynamic Event Placeholders:** Built-in scheduling architecture ready to populate senior-led events, currently hosting a modern placeholder notification.
- **Logistics & Team Registry:** A dedicated overview space mapping the operational coordinators supporting our community leaders behind the scenes.
- **Aesthetic Typography:** Structured hierarchy featuring crisp navy accents (`#003366`) and balanced, accessible reading elements optimized for all generations.

## 📂 Project Architecture

The repository is built entirely on fundamental front-end web tech (Vanilla HTML5 & CSS3) ensuring lightning-fast load times, exceptional accessibility, and zero external engine dependencies:

```text
ngowebsitepageproject/
│
├── index.html       # Homepage showcasing the baseline hero identity
├── about.html       # The operational core, structural missions, and objectives
├── events.html      # Community bulletin interface tracking senior-led sessions
├── team.html        # Grid profile space displaying organizational staff
├── style.css        # Core design engine handling grids, responsive media queries, and themes
└── script.js        # Baseline client-side lifecycle and operational logs
